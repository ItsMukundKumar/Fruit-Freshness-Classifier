<h1 align="center">🍎 Fruit Freshness Classifier</h1>

<p align="center">
A Deep Learning project that classifies fruit images as <b>Fresh</b> or <b>Rotten</b> using a lightweight CNN model and a Streamlit web app.
</p>

<p align="center">
🚀 TensorFlow Lite • 🎯 Binary Classification • 🎨 Streamlit UI
</p>

---

<h2>🚀 Project Overview</h2>

This project uses a <b>Convolutional Neural Network (CNN)</b> to classify fruit images into:

<ul>
    <li>✅ Fresh</li>
    <li>🥀 Rotten</li>
</ul>

The model has been optimized and converted to <b>TensorFlow Lite (.tflite)</b> for faster performance and efficient deployment.

---

<h2>⚡ Key Highlights</h2>

<ul>
    <li>📦 Lightweight <b>TFLite model</b> (fast loading)</li>
    <li>⚡ Optimized for deployment</li>
    <li>🚀 No external downloads required</li>
    <li>🎯 Real-time prediction</li>
</ul>

<h2>📦 Installation & Setup</h2>

<h3>1️⃣ Clone Repository</h3>

<pre>
git clone https://github.com/ItsMukundKumar/Fruit-Freshness-Classifier.git
cd Fruit-Freshness-Classifier
</pre>

<h3>2️⃣ Install Dependencies</h3>

<pre>
pip install -r requirements.txt
</pre>

<h3>3️⃣ Run Application</h3>

<pre>
streamlit run main.py
</pre>
