<h1 align="center">🍎 Fruit Freshness Classifier</h1>

<p align="center">
A Deep Learning project that classifies fruit images as <b>Fresh</b> or <b>Rotten</b> using CNN and a Streamlit web app.
</p>

<p align="center">
🚀 TensorFlow • 🎯 Binary Classification • 🎨 Streamlit UI
</p>

---

<h2>🚀 Project Overview</h2>

This project uses a CNN model to classify fruit images into:
<ul>
<li>✅ Fresh</li>
<li>🥀 Rotten</li>
</ul>
